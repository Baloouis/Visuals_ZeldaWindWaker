Recreating my favorite visuals, VFXs and shaders from Zelda The Wind Waker.

# Current Features : 
- Toon Water shader
- Waves Foam VFX
- Wind trails VFX
- Toon Explosion VFX
- Bomb Explosion Build-up Animation
